# Opinion-Extraction

## Requirements

- Python 3.6
- pyhanlp 


_pyhanlp_(avaliable at **[here](https://github.com/hankcs/pyhanlp)**) is used to extract the NER.


If you want to use other tools, please rewrite SpeakExtraction.get_names()


## Usage
